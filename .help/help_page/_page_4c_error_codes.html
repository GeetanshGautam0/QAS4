FOUR CHARACTER ERROR CODES

4 character error codes are those that follow the aTSP codes and are located in square brackets. These codes specify the general class of the exception.

Errors 0x1000 to 0x1FFF (FileIO-Related Errors)

0x13EC

CASE 1: EOFError

End of File (EOF) errors occur when an interpreter reaches the end of a file's content unexpectedly/prematurely. Can be raised by:

  • The PYTHON interpreter (an internal error)
  • The QUIZZING APP FILE PARSER

0x13F6

CASE 1: NotADirectoryError

Not a Directory errors occur when a file IO function attempts to run a directory-specific command on an object that isn't a directory. Can occur if:

  • The object is not a directory (is a file)
  • The object does not exist

0x1400

CASE 1: IsADirectoryError

Is a Directory errors occur when a file IO function attempts to run a non-directory command on an object that is a directory. Can occur if:

  • The object is a directory (if trying to run a FileIO task (R/W) on an object that is a directory)
  • The object does exist (if trying to create a directory that already exists, for example)

0x141E

CASE 1: FileNotFoundError

File Not Found errors occur when a file IO function attempts to run a file-specific command on an object that does not exist. Can occur if:

  • The object is not a file
  • The object does not exist

0x15E0

CASE 1: FileExistsError

File Exists errors occur when a file IO function attempts to run a command on a file object that does exist. Can occur if:

  • The object does exist (if a file IO function is trying to create a file that already exists.)

Errors 0x2000 to 0x2FFF (General Errors)

0x23EA

CASE 1: TypeError

Type errors occur when the wrong type of data is passed to a function.

  • An internal error

0x23F3

CASE 1: AttributeError

Attribute errors occur when unexpected data is passed to a function.

  • An internal error

0x23F4

CASE 1: NameError

Name errors occur when an undeclared variable is accessed.

  • An internal error

0x244D

CASE 1: ValueError

Same as 23F3 Attribute Error.

Errors 0x3000 to 0x3FFF (General Python Errors - P0 TYPE ERRORS)

0x33F2

CASE 1: Exception

A generic exception.

0x33FC

CASE 1: Base Exception

A BaseException type error is raised anytime a (custom) exception defined and raised by the quizzing application suite (such as qa_def.NotYetImplemented).

CASE 2: BlockingIOError

An internal, python error (P0).

CASE 3: BrokenPipeError

An internal, python error (P0)

CASE 4: BufferError

An internal, python error (P0)

CASE 5: ChildProcessError

  • An internal, python error (P0)
  • Can occur if a subprocess called by the quizzing app suite (such as the theme download prompt) encounters an error and does not catch it.

CASE 6: ConnectionAbortedError

An internal, python error (P0)

CASE 7: ConnectionError

An internal, python error (P0)

CASE 8: ConnectionRefusedError

An internal, python error (P0)

CASE 9: ConnectionResetError

An internal, python error (P0)

CASE 10: ImportError

An internal, python error (P0); can occur if (when in development mode), a dependency of the application is not met. Check PYTHON DEPENDENCIES fore more information.

CASE 11: IndentationError

An internal, python error (P0)

CASE 12: IndexError

An internal, python error (P0); occurs when a list/tuple/set is accessed at an index that does not exist.

CASE 13: KeyError

An internal, python error (P0); occurs when a dict is accessed with a key that it does not contain.

CASE 14: InterruptedError

An internal, python error (P0). KEYBOARD INTERRUPTS ARE NOT CAUGHT BY THE GLOBAL ERROR MANAGER BY DESIGN.

CASE 15: LookupError

An internal, python error (P0)

CASE 16: ModuleNotFoundError

An internal, python error (P0); akin to CASE 10: ImportError.

CASE 17: NotImplementedError

An internal, python error (P0)

CASE 18: OverflowError

An internal, python error (P0)

CASE 19: ProcessLookupError

An internal, python error (P0)

CASE 20: ReferenceError

An internal, python error (P0)

CASE 21: TabError

An internal, python error (P0)

CASE 22: TimeoutError

An internal, python error (P0)

CASE 23: UnboundLocalError

An internal, python error (P0)

CASE 24: UnicodeError

An internal, python error (P0); can occur when trying to access a file that was created with a different encoding (different locale).

CASE 25: UnicodeWarning

An internal, python error (P0)

CASE 26: UnicodeDecodeError

An internal, python error (P0)

CASE 27: UnicodeEncodeError

An internal, python error (P0)

CASE 28: UnicodeTranslateError

An internal, python error (P0)

0x33F3

CASE 1: AssertionError

Occurs when unexpected data is encountered by a function, or when essential functions fail.

0x33F4

CASE 1: PermissionError

Occurs when an operating-system level command is run without sufficient permissions. Usually caused by FileIO functions.

0x33F6

CASE 1: RecursionError

Occurs when a recursive function cycles too many times.

0x33F7

CASE 1: SystemError

Results from a failure stemming from the operating system API (SYS and OS modules in python)

0x33FF

CASE 1: Memory Error

An internal, python error (P0).

0x3400

CASE 1: RuntimeError

An internal, python error (P0).

0x3401

CASE 1: Runtime Warning

An internal, python error (P0).

0x3402

CASE 1: Syntax Warning

An internal, python error (P0).

0x3403

CASE 1: Syntax Error

An internal, python error (P0).

0x3410

CASE 1: IOError

An internal, python error (P0) -- these errors are not raised by the quizzing application FileIO handler.

CASE 2: EnvironmentError

An internal, python error (P0).

CASE 3: OSError

An internal, python error (P0).

CASE 4: WindowsError

An internal, python error (P0) -- same as OSError on Windows machines.

Errors 0x4000 to 0x4FFF (Arithmetic Errors)

0x43F3

CASE 1: Floating Point Error

Occurs when a floating-point number encounters an error (P1 type).

0x44BD

CASE 1: ZeroDivisionError

Occurs when a division by zero event occurs; usually the result of invalid array computations (P1 type).

0x45E6

CASE 1: Arithmetic Error

General Arithmetic Errors (P1 type).
